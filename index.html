Table Tags
Tag Sintax Description
Table <table> Table content goes here </table> All table content must go between this tag.
Caption <caption> Text goes here </captionr> It can be used to title a table.
Table Head <thead> Table head content goes here </thead> Semantic tag to mark the table head.
Table Header <th> Text goes here </th> It can be used to name the table colums.
Table Body <tbody> Table body content goes here </tbody> Semantic tag to mark the table body.
Table row <tr> Table row content goes here </tr> It can be used to add rows to a table.
Table Data <td> Table cell content goes here </td> It can be used to add cells to a table.
Table Foot <tfoot> Table footer content goes here </tfoot> It can be used to mark sumarized data in the table.

Table Attributes
Attribute Sintax Description
Colspan colspan='number' Specifies the number of colums a cell will take
Rowspan rowspan='number' Specifies the number of rows a cell will take

Docker Commands
Command Sintax Description
docker run docker run -d -p --name image_id container_name Search for a given image to run a container.
docker build docker build -t image_name . Builds an image out if a docker file.
docker images docker images Shows the images built on the host.
docker container docker container ls Shows running containers.
docker container ps -a Shows running and stopped containers.
docker exec docker exec -it container_id bash Logs into a given container cli.
exit exit Logs out of a container cli.
docker logs docker logs container_id Shows the logs of a given container.
docker stop docker stop container_id Stops a given container.
docker start docker start container_id starts a given container.
docker rm docker rm container_id Removes a given container.

Docker flags
Sintax Description
-t image_name Names a given image.
-d image_id Allows to run other commands while the container is built in the background.
-p host_port:container_port Opens a given host/container port.
--name container_name Names a given container.
