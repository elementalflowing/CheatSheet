Docker

Docker Commands
Command Sintax Description
docker info docker info Display system wide information.
docker run docker run -d --name container_name -p host_port:container_port -v named_volume:path image_id Search for a given image to run a container.
docker build docker build -t image_name . Builds an image out if a Dockerfile.
docker images docker images Shows the images built on the host.
docker image docker image history image_id Shows the layer history of an image.
docker container docker container ls Shows running containers.
docker container ps --no-trunc -a Shows running and stopped containers.
docker container ps -s Shows the size of the running containers.
docker exec docker exec -it container_id my_shell Executes a given command into a running contanier.
exit exit Logs out of a container cli.
docker logs docker logs container_id Shows the logs of a given container.
docker stop docker stop container_id Stops a given container.
docker start docker start container_id Starts a given container.
docker rm docker rm -f container_id Removes a given container.
docker image rm image_id Removes a given image.
docker login docker login -u my_user_name Logs in to Docker Hub passing a given username.
docker tag docker tag image_id my_user_name/new_image_name Tags a given image with a name to be pushed.
docker push docker push my_tag Push a given tagged image to a registry.
docker volume docker volume ls Shows a list of all volumes available.
docker volume rm volume_id Removes a given volume.
docker volume prune Removes all unused volumes.
docker volume create volume_id Creates a named volume.
docker-compose docker-compose up -d Runs a docker-compose file in detached mode.
docker-compose start Starts containers in a docker-compose file if they are already created.
docker-compose stop Stops containers in a docker-compose file.
docker-compose down --volumes Stops and romoves containers in a docker-compose file along with the volumes.
docker-compose logs -f Shows the logs as they are generated.
docker network docker network create network_name Creates a network.

Docker flags
Sintax Description
-t image_name Names a given image.
-u my_user_name Specifies a given username lo log in.
-d image_id Allows to run other commands while the container is built in the background.
-f container_id Force a container to be removed if it hasn't been stopped first.
-p host_port:container_port Opens a given host/container port.
--name container_name Names a given container.
-v named_volume:path Specifies a path to map a named volume in a container.
-it container_name Allows the container to run with an interactive prompt.
--no-trunc container_name Not truncated output.
--fliter "name=container_name" Fliter a given container by name.
